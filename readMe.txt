Para compilar tudo basta digitar "make" na consola, na respetiva pasta.

Com o conteúdo atual da função main no interpreter.c, na shell é printado o código de 3 endereços, e o respetivo código MIPS é enviado para um ficheiro MIPS.txt que será criado aquando a execução do programa.

Juntamente nesta pasta, serão enviados três exemplos, que podem ser testados com "./interpreter exemplo1.txt", trocando o número do exemplo para o pretendido.
